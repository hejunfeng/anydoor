# anydoor

Tiny NodeJS Static Web Server

## 安装

npm i -g anydoor

## 使用方法
anydoor # 把当前文件夹当做静态资源服务器根目录

anydoor -p 8080 # 设置端口号 8080

anydoor -h localhost # 设置host为locahost

anydoor -d /D: # 设置根目录为D:
